DEBIAN

Debian adalah distribusi sistem operasi berbasis Linux yang dikembangkan oleh komunitas. Ini dikenal dengan pendekatan kuat terhadap prinsip perangkat lunak sumber terbuka, keamanan, dan stabilitas. Debian berfokus pada kebebasan perangkat lunak dan memberikan infrastruktur untuk berbagai arsitektur dan lingkungan. Ini termasuk manajemen paket yang kuat menggunakan alat APT (Advanced Package Tool) untuk instalasi dan pembaruan perangkat lunak. Debian juga dikenal sebagai basis untuk berbagai distribusi Linux lainnya.
